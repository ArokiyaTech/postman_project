Hands-on Postman API testing project showcasing manual testing skills for BCA freshers. Contains exported collections with GET/POST requests to public APIs like GoRest, test scripts, environments, and Newman reports for CI/CD integration.

Project Overview
Repository documents practical Postman experience through real-world API testing workflows, including authentication (Bearer tokens), CRUD operations, and response validation. Ideal for fresher resumes to demonstrate API testing proficiency without coding expertise.
​

Key Features
Collections: GET user data (/public/v2/users/7733563), POST new users, auth headers setup.
​

Environments: Variables for base URLs, tokens from GoRest login.
​

Tests: Chai assertions for status codes, JSON validation, response times.
​

Newman Scripts: Automated runs with HTML reports for GitHub Actions.
​

README Guide: Step-by-step import, execution, troubleshooting for beginners.
​

Tech Stack
Postman Collections (JSON), Newman CLI, GitHub Actions, Public REST APIs (GoRest.co.in). Clone, import to Postman, and run to replicate testing journey. Perfect for BCA students building QA portfolios.
